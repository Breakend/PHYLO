Contribute to Phylo!

Phylo is an open-science project. That means that anyone can contribute either by coding, translating, submitting or analyzing the data... and of course playing!

Code

Phylo is open-source and we rely on you to improve it! This is non-profit project that aims to provide to the scientific community a new tool to analyze genomes. It has been developed by McGill student volunteers. We need you to make it better and more fun!

Translate

You don't know how to code? Don't' worry, you can still contribute. We aim to distribute Phylo in any language possible. We need your help to translate it. You can either create a new translation or improve an existing one. All you need to do is to translate one file.

Analyze the data

By playing Phylo, gamers are generating many new DNA alignments. We filtered the most promising data and make them available to anyone who is willing to have a look at it. Download the alignments and improve the annotation of the human genome.
